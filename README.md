# **Plinko**
*Progetto Java 2025/2026 Odierna Thomas, Alberto Sambruni*  
Creazione gioco d'azzardo digitale "Plinko"
## Inizio progettazione
https://plinko-game-online.github.io/
1) Creazione schermo: 15 righe, 3 "ostacoli" iniziali, aumentano di 1 ogni riga
2) Creazione pallina: raggio = 60% distanza in x tra gli ostacoli adiacenti sulla stessa riga, puntata affidata alla pallina
3) Funzioni pallina: gravità, rimbalzo.
4) Gravità
5) Rimbalzo
6) Generazione pallina: generata in posizione casuale tra i primi tre ostacoli
7) Arrivo con moltiplicatori
8) Salvataggio valori su file txt di: Vittoria o perdita, guadagno, soldi nel portafoglio e moltiplicatore raggiunto
